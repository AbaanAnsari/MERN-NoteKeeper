## 💡 About

**MERN-NoteKeeper** is a modern web application that helps users manage notes efficiently. It demonstrates a full MERN (MongoDB, Express.js, React, Node.js) implementation with a clean frontend and a RESTful backend API.

The application supports the core **CRUD** operations:
- 🆕 Create notes
- 👀 View all notes
- ✏️ Edit/update notes
- 🗑️ Delete notes

---

| Layer | Technology |

| Backend | Node.js + Express.js |
| Database | MongoDB |

## 🚀 Features

✨ Full Create, Read, Update, Delete (CRUD) functionality  
✨ Responsive UI for managing notes  
✨ Separation of frontend and backend  
✨ Uses RESTful APIs for client-server communication  
✨ Deployed and production-ready setup
