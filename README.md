## 💡 About

**MERN-NoteKeeper** is a modern web application that helps users manage notes efficiently. It demonstrates a full MERN (MongoDB, Express.js, React, Node.js) implementation with a clean frontend and a RESTful backend API.

The application supports the core **CRUD** operations:
- 🆕 Create notes
- 👀 View all notes
- ✏️ Edit/update notes
- 🗑️ Delete notes
